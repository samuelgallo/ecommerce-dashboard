# Simple ecommerce dashboard

Node.js, Express, Mongoose (MongoDB), EJS

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

## Docker Compose

Run with Docker

```bash
docker-compose up
```

[localhost:3000/dashboard](localhost:3000/dashboard)
